# Paperclip Stop Condition Escalation

- Do not stop for ordinary implementation failures.
- Escalate to bounded investigation when repeat failure exposes a deeper boundary problem.
- Allow only mission-preserving re-plans.
- Surface human intervention only for true deployment or authorization blockers.
